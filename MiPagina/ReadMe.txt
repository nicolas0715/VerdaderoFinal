Descarga
    Para descargar el repositorio ejecutar lo siguiente en la terminal:
        Situado en la carpeta del proyecto:
            git clone https://github.com/nicolas0715/VerdaderoFinal.git
        Una vez descargado, ejecutar el siguiente comando:
            python manage.py runserver
        Luego ingresar en la web con el link que aparece en la terminal:
            ejemplo: http://127.0.0.1:8000/

Navegacion
